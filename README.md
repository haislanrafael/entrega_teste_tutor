


# Roguelike Adventure (PgZero)

This project is a small roguelike-style game developed using **Python** and **PgZero (Pygame Zero)**.

The goal of this project was to demonstrate core game development concepts such as grid-based movement, enemy AI behavior, collision detection, game states, and audio integration.

---

## 🎮 Game Overview

The player controls a hero that moves across a grid-based map.  
An enemy moves automatically within a defined territory.  

If the player collides with the enemy, the game ends.

The game includes:

- Main menu
- Sound toggle system
- Smooth grid-based movement
- Enemy automatic patrol behavior
- Collision detection
- Game over screen
- Restart system
- Background music support

---

## 🧠 Technical Concepts Implemented

- Object-Oriented Programming (OOP)
- Grid-based movement system
- State management (`menu`, `playing`, `game_over`)
- Basic AI movement logic (enemy patrol)
- Collision detection using logical grid positions
- Smooth position interpolation
- Audio management with `pygame.mixer`
